# Frontend Modules
Navigation badges updated via service subscriptions.

Panels:
- NationsPanel: formation assignment
- FormationsPanel: categories + sub-formation links
- UnitEditor: full unit graph (stats, capabilities, grenades, guns, ammo, fire modes, equipment)
- TemplatesPanel: weapons, ammo templates, fire modes, tags
- StatsPanel: summary metrics (charts roadmap)
- SettingsPanel: theme prefs (future persistence)

Styling: SCSS tokens; keep theme variables centralized.
